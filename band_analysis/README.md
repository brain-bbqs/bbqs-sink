# Band Analysis

Though analysis of oscillations is a very complex topic in neuroscience, this collection contains a few small utilities that have been used in older projects.

This code has been adapted from the [Neuroscope utils](https://github.com/catalystneuro/buzsaki-lab-to-nwb/blob/master/buzsaki_lab_to_nwb/utils/band_analysis.py) of the [Buzsaki lab](https://buzsakilab.com/wp/) at NYU.



### Installation

Using any Python setup, install the minimal dependencies with:

```bash
pip install numpy scipy
```
